# Ostrom Home Assistant Integration

Custom Home Assistant integration for Ostrom dynamic electricity prices.

## Features
- OAuth2 client credentials flow
- Token caching
- Spot price sensor
- Home Assistant config flow

## Installation
1. Add this repository to HACS as a custom repository.
2. Install the integration.
3. Restart Home Assistant.
4. Add the Ostrom integration from the UI.

## Configuration
- Client ID
- Client Secret
- Zip Code
- Environment
- Endpoint

## Notes
This integration requires an active Ostrom contract.
